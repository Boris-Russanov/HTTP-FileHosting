Welcome to my instance of JserveMe!

Click the "data" folder to expand it and view its contents.

To download the files you see, simply click on the text of the file, or right click for dropdown menu.

You can also get the file by entering the path relative to "data/" and the file at the end (e.g: folder/file).

File adding is avaiable! Choose which file to upload below by clicking "choose file".

Note: You can not download the whole folder but only individual files at the moment.

Note: Expect a delay for bigger file uploads, updates are indicated via "data" folder refresh.

Enjoy!
